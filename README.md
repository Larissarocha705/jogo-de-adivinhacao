## 📌 Sobre o projeto

Este é um jogo de adivinhação desenvolvido em linguagem C, onde o jogador precisa descobrir um número secreto gerado aleatoriamente.

O projeto tem como objetivo praticar:

- Estruturas condicionais (`if`, `else`)
- Laços de repetição (`while`, `for`)
- Lógica de programação
- Entrada e saída de dados

---

## 🎮 Como funciona

- O jogo inicia com uma tela de boas-vindas
- Um número secreto é gerado automaticamente
- O jogador tenta adivinhar o número
- O programa informa se o palpite é:
  - 🔼 Maior que o número secreto
  - 🔽 Menor que o número secreto
- O jogo termina quando o jogador acerta

---

## 💻 Tecnologias utilizadas

- Linguagem C
